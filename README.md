# Trainee Management System

## Overview

This project was developed during my Cooperative Training using ASP.NET Web Forms.

The system is designed to simplify trainee management by providing features for user registration, login, clearance request submission, and administrative management through a web-based interface connected to a SQL Server database.

## Features

- User Registration
- User Login & Authentication
- Clearance Request Form
- Secure File Upload
- Server-side Validation
- Administrative Dashboard
- SQL Server Database Integration
- Responsive User Interface

## Technologies Used

- ASP.NET Web Forms
- C#
- Microsoft SQL Server
- HTML5
- CSS3
- JavaScript
- Bootstrap
- Visual Studio

## Getting Started

1. Clone the repository.
2. Open the solution in Visual Studio.
3. Restore the required NuGet packages.
4. Update the SQL Server connection string in `Web.config`.
5. Build and run the project.
