# Trainee Management System

## Overview

This project was developed during my Cooperative Training using ASP.NET Web Forms.

The system helps manage trainee information through a web application that includes user registration, login, clearance request submission, and administrative features. It also uses SQL Server to store and manage data.

## Features

- User Registration
- User Login
- Clearance Request Form
- File Upload
- Form Validation
- Admin Dashboard
- SQL Server Database Integration

## Technologies Used

- ASP.NET Web Forms
- C#
- SQL Server
- HTML
- CSS
- JavaScript
- Bootstrap

## My Responsibilities

As part of the development team, I worked on:

- Enhancing the Clearance Form.
- Improving the user interface.
- Implementing server-side validation.
- Developing the file upload functionality.
- Integrating the application with SQL Server.
- Testing and debugging the system.

## How to Run

1. Clone the repository.
2. Open the solution in Visual Studio.
3. Restore the required NuGet packages.
4. Update the connection string in `Web.config`.
5. Build and run the project.
