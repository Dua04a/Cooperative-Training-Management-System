# Cooperative Training Management System

## About

This project was developed during my cooperative training as part of a larger training management system.

I was responsible for developing the Clearance module, which allows trainees to submit clearance requests, upload the required documents, and securely store the submitted information in the database.

This repository contains the student version of the project. It was created for educational and portfolio purposes only. It was not deployed as an official production system, and all organizational branding and sensitive information have been removed.

---

## Technologies

- ASP.NET Web Forms
- C#
- SQL Server
- HTML
- CSS
- JavaScript
- Visual Studio

---

## Features

- Clearance request form
- File upload with validation
- SQL Server database integration
- Required field validation
- Arabic and English language support
- Responsive user interface
- Secure database insertion using parameterized queries

---

## My Contribution

This project was developed as part of a team during my cooperative training.

My responsibilities included:

- Designing and developing the Clearance module.
- Building the frontend interface using HTML, CSS, and JavaScript.
- Developing the backend logic using C# and ASP.NET Web Forms.
- Creating and connecting the SQL Server database for the Clearance module.
- Implementing file upload functionality with validation.
- Adding Arabic and English language support.
- Testing and debugging the module before the final system integration.

## How to Run

1. Clone the repository.
2. Open the solution in Visual Studio.
3. Restore the required packages.
4. Configure the SQL Server connection string.
5. Create the database using the provided SQL script.
6. Run the project using IIS Express.
